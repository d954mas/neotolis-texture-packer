# 0001 — PUC Lua 5.5 для sandboxed format handlers

**Дата:** 2026-07-12
**Статус:** accepted
**Принял:** owner (d954mas), подтверждено в обсуждении kickoff
**Реализуется в:** B3-01 (master spec §32, §45–46, §59 items 33–36)

## Решение

Sandboxed Lua runtime строится на PUC Lua 5.5 (MIT), не LuaJIT и не 5.4.

Условия:

- пиним последний patch release на момент B3-01 и накатываем официальные
  багфиксы с lua.org bug list;
- загрузка binary chunks запрещена (mode `"t"` only);
- обязательный adversarial sandbox suite до автоматического исполнения
  external packages (master spec §32 Automatic sandboxed execution);
- fallback: если к моменту B3-01 spike в 5.5 подтверждается активный поток
  багов в областях, которые мы используем (GC, debug hooks, allocator),
  откатываемся на последний PUC Lua 5.4.x — API sandbox-слоя не меняется.

## Контекст

Lua 5.5.0 вышла 22.12.2025; на 2026-07 активно поддерживается (bugfixes
5.5.0-6/7 закрыты в июне 2026). B3 наступит после C0/F1/F2/B0/B1/F3/B2, к этому
моменту 5.5 станет заметно зрелее. Выбор 5.5 сразу избегает миграции 5.4→5.5
посреди жизни format API (§31: один current `format_api_version` на релиз).
