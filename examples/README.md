# Example projects

These projects are portable, checked-in `.ntpacker_project` files that can be
opened directly in `ntpacker-gui` or used with the CLI. Keep each project next
to the rest of this `examples/` directory: its source paths are relative.

| Project | Purpose |
|---|---|
| `showcase/showcase.ntpacker_project` | Small visual demo over 60 CC0 animal sprites. Start here. |
| `projects/large-synthetic.ntpacker_project` | Performance and recovery fixture: 100 atlases, 1,000 file-source memberships, 100 export targets. |
| `defold-demo/defold-demo.ntpacker_project` | Real Defold comparison project with reference and ntpacker exports. |

Release archives include this directory. From a source checkout, the same
files are available without generation.
